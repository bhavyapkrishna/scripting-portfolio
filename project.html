PHP Scripting Manual

Scripting Manual

I used PHP to make a web page that allows the user to look at only specific sections of scripting manuals available on the man7 manual directory.
If the manual and/or section doesn't exist, it will notify the user that the title/section is invalid. Otherwise, it will print out the correct one.
The images show the app, what the input manual/section are, and the section output.
